# iSport - Keep and Lrean !

## You can develop the project by the following steps:

Install yalc:
```sh
npm i -g yalc
npm i -g yarn
```

cd into project's folder:
```sh
cd posenet
```

Install dependencies:
```sh
yarn
```

Build the project:

```sh
yarn build
```

Start the dev demo server:

```sh
yarn watch
```
